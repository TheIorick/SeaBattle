9. Морской бой. Необходимо реализовать механику игры в морской
бой с использованием мин, подводных лодок, минных тральщиков.

Морской бой v.1
Четыре поля 10*10
	1. Моё поле с кораблями
	2. Поле противника по которому я стреляю
	3. Поле с кораблями противника
	4. Моё поле для противника по которому он стреляет
На поле можно разместить (для поля 10*10): 
Корабли:
 4 - однопалубника (корабли в одну клетку)
 3 - двухпалбника (корабли в две клетки)
 2 - трехпалубника (корабли в три клетки)
 1 - четырехпалбуник (корабль в 4 клетки или 2*2 клетки)
ДОПОЛНИТЕЛЬНО - мину, минный тральщик, подводная лодка
Как работает:
Мина (кружочек вписанный в клетку) - при попадании на мину, игрок, который делал выстрел должен раскрыть координаты одной клетки

 на которой находится часть какого-либо его корабля, по которой ещё не был произведен выстрел, своему противнику (хозяину мины), которому также дается дополнительный ход. 

Минный тральщик (равнобедренный треугольник) - при попадании в минный тральщик, игрок, который делал выстрел, должен раскрыть координаты одной клетки на которой находится мина, по которой ещё не был произведен выстрел, если таковые имеются. 

Подводная лодка представляет собой 3х палубный корабль, который тонет при первом попадании в него. Для отличия он будет заштрихован по краям внутренних клеток.

Как размещать корабли?
	Корабли нельзя размещать рядом друг с другом вплотную, между ними должно быть расстояние в минимум одну клетку. Их можно размещать только по вертикали и горизонтали. Аналогично для доп.юнитов. 
	Корабли помечаются при попадании, что их ранили (кроме однопалубных, мин, тральщика и подводной лодки, которые сразу отмечаются как пораженные).
	Когда у корабля не остается целых клеток (когда есть попадания по всем клеткам корабля) он взрывается, задевая взрывной волной все клетки граничащие с ним.
Они отмечаются как клетки, по которым уже был произведен выстрел.

Ход игры
Игроки ходят по очереди, называя координаты, по которым они хотят произвести выстрел. После выстрела происходит следующее:
---Пустая клетка: отмечается, как пораженная. Стрелявшему сообщается - МИМО
---Клетка с кораблем: отмечается, как пораженная. Если корабль > 1, то сопернику сообщается - РАНИЛ, иначе УНИЧТОЖИЛ и отмечаются все клетки вокруг уничтоженного корабля.
---Клетка с миной: отмечается, как пораженная. Стрелявшему сообщается - МИНА! Стрелявший должен выдасть координаты одной клетки на которой находится часть какого-либо его корабля, по которой ещё не был произведен выстрел. У его противника есть доп.ход. 
---Клетка с тральщиком: отмечается, как пораженная. Стрелявшему сообщается - ТРАЛЬЩИК! Стрелявший должен раскрыть координаты одной клетки на которой находится мина, по которой ещё не был произведен выстрел, если таковые имеются.
---Клетка с подлодкой: Стрелявшему сообщается - ПОПАЛ В ПОДВОДНУЮ ЛОДКУ. Вся подлодка отмечается как пораженная со всеми клетками, граничащими с ней.
Если стрелявший промазал, то ход переходит к другому игроку, иначе игрок стреляет ещё раз

Игра продолжается до тех пор, пока, у кого-либо из игроков не потонут все корабли и подводная лодка. Игрок у которого остались целые корабли и/или подлодка объявляется победителем.
